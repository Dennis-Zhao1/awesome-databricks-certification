# Contributing

Thank you for helping to improve this list!

## Quality Standards

Resources must meet **all** of the following:

- **Relevant** — Directly related to at least one Databricks certification exam
- **Current** — Content aligned to the current exam guide (2024 or later)
- **English** — Written primarily in English
- **Accessible** — Content is publicly available (free or clearly marked as paid)
- **Original** — Not a fork, clone, or copy of an existing resource on this list
- **Non-violating** — No exam dumps, NDA-violating materials, or brain dumps

## How to Add a Resource

1. Fork this repository
2. Add your resource to the appropriate certification section
3. Use the format: `- [author/repo-name](URL) - One-line description of what this resource provides.`
4. Place the entry in the correct subsection (Books, Courses, Study Notes, Hands-On Labs, Study Resources)
5. Submit a pull request with a brief explanation of why this resource is useful

## How to Remove a Resource

If a resource is broken, archived, outdated, or no longer meets quality standards:

1. Open an issue describing the problem, or
2. Submit a pull request removing the entry

## Link Format

- GitHub repos: `[author/repo-name](https://github.com/author/repo-name)`
- Websites: `[Descriptive Title](https://example.com)`
- Descriptions should be one sentence, ending with a period
- Do not include star counts or other volatile metadata
